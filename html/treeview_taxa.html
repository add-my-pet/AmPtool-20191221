treeview_taxa

shows pedigree in interactive html

Contents

  • Syntax
  • Description
  • Example of use

Syntax

treeview_taxa (taxon)

Description

First procudes pedigree with function pedigree and uses the result to create file /treeview/treeview_taxa.js and opens AmPtool/taxa/treeview/treeview_taxa.html in the system browser.

Input:

  • taxon: character string with a taxon that is a node in the taxonomic tree, see list_taxa

Example of use

treeview_taxa('Cladocera');


Published with MATLAB® 7.14
